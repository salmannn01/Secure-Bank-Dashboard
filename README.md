# Secure-Bank-Dashboard
SecureBank Dashboard 🏦
A modern, responsive, single-page online banking dashboard interface designed with Tailwind CSS and powered by vanilla JavaScript.

🚀 Features
Live Balance Updates: Displays real-time calculations for Total, Checking, and Savings balances based on transactions.

Interactive Transaction Modals: Functional popup windows for processing Money Transfers, Bill Payments, and Deposits.

Dynamic Ledgers: Automatically lists recent transaction logs with contextual green (+) and red (-) color indicators.

Reactive UI: Leverages clean CSS animations, slide-in toasts, responsive mobile grid layouts, and smooth modal transition toggles.

🛠️ Architecture & Technologies
Frontend Structure: HTML5 Semantic Markup

Styling Framework: Tailwind CSS v3 via CDN

Typography: Inter font family integration

Logic Layer: Vanilla JavaScript (DOM manipulation, FormData API, asynchronous timers for components)

📂 Quick Start
Save the code file as index.html.

Open the file directly in any modern web browser (e.g., Chrome, Safari, Firefox).

No local compilation or server environments are required.
